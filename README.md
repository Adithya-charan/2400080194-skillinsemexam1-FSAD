## Project Overview

A Maven + Hibernate project demonstrating **Hibernate Query Language (HQL)** operations on a `Ticket` entity.

### Operations Implemented

| # | Operation | Details |
|---|-----------|---------|
| I | **Insert** records | Using persistent object via `session.save()` |
| II | **Update** `Name` & `Status` by `ID` | HQL with **positional parameters** (`?1`, `?2`, `?3`) |

## Ticket Entity

| Column | Type | Description |
|--------|------|-------------|
| `ticket_id` | INT (PK, Auto) | Primary Key |
| `ticket_name` | VARCHAR(100) | Ticket title |
| `ticket_date` | DATE | Created date |

## HQL Operations

### I. Insert using Persistent Object
```java
// Ticket becomes persistent when saved within a session
Transaction tx = session.beginTransaction();
session.save(ticket);  // INSERT SQL generated by Hibernate
tx.commit();
```

### II. Update Name & Status by ID — Positional Parameters
```java
String hql = "UPDATE Ticket t SET t.name = ?1, t.status = ?2 WHERE t.id = ?3";
Query<?> query = session.createQuery(hql);
query.setParameter(1, newName);     // positional param ?1
query.setParameter(2, newStatus);   // positional param ?2
query.setParameter(3, ticketId);    // positional param ?3
int rows = query.executeUpdate();
```

## Setup Instructions

### Prerequisites
- Java 11+
- Maven 3.6+
- MySQL 8.0

### 1. Create Database
```sql
CREATE DATABASE fsadexam;
```

### 2. Configure DB credentials
Edit `src/main/resources/hibernate.cfg.xml`:
```xml
<property name="hibernate.connection.username">root</property>
<property name="hibernate.connection.password">root</property>
```

### 3. Build & Run
```bash
mvn clean compile
mvn exec:java -Dexec.mainClass="com.klef.fsad.exam.ClientDemo"
```

> Hibernate will auto-create the `tickets` table on first run (`hbm2ddl.auto=update`).
